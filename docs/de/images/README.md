# Deutsche Screenshots

Dieser Ordner enthält **deutschsprachige Screenshots** für die Dokumentation.

## Verwendung in Markdown:

```markdown
![Screenshot Beschreibung](images/dateiname.png)
```

## Naming Convention:

- Beschreibend und kurz
- Kleinbuchstaben mit Unterstrichen
- Beispiele:
  - `quickstart_01_service_auswahl.png`
  - `ui_hauptseite.png`
  - `config_download.png`
