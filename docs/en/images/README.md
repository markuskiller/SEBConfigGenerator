# English Screenshots

This folder contains **English language screenshots** for documentation.

## Usage in Markdown:

```markdown
![Screenshot description](images/filename.png)
```

## Naming Convention:

- Descriptive and concise
- Lowercase with underscores
- Examples:
  - `quickstart_01_service_selection.png`
  - `ui_main_page.png`
  - `config_download.png`
